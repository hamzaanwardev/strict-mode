Strict Mode

In JavaScript the Strict mode will help us to debug an error in JavaScript.

in strict mode the undeclared variable can not be executed.

For strict mode in js we only have to write a string that is 'use strict'.

In the below example, when i disable the strict mode, it will not show any error for any undeclared variable. But when i enable it it will show error in console for undeclared variable.

Strict Mode also gives error for js reserved words for future.
